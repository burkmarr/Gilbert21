Some queries saved with version 1 don't work with version 2

This is because Gilbert moved from and Access database to a SQLite database and these have a different SQL query syntax. Just bin the old queries and build new ones. To a large extent, this problem has been superseded by the addition of several new, simpler, record selection methods.
